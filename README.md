# Mail_familiarSpirits

## 邮件使魔

### 安装依赖
> composer require phpmailer/phpmailer

### 使用说明

* 接收信息在本地记录日志并且发送到你的邮箱
* `config.php`配置你的`smtp`有关信息
* 访问路径demo `/sendMail.php?message=Hello%20World` ,从`message`传入数据
